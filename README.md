# My-Zootopia-with-API
New training Repo to extend My-Zootopia with API functionality

This project asks the user to enter an animal name and fetches all 
the data from the API https://api-ninjas.com/api/animals.
The results get parsed into an html template and a simple website
gets generated.

## Installation

To install this project, simply clone the repository and install the dependencies in requirements.txt using `pip`

## Environment

Create an .env file in the root directory of the project and insert 
the API Key you gathered from https://api-ninjas.com/ like:
API_KEY='YOUR_API_KEY'

## Usage

To use this project, run the following command - `python animals_webgenerator.py`.
